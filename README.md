# Quizlet Downloader
Scraping content from Quizlet and saving it locally.

## Setup
1. Clone or download this repo as zip then extract
2. Open browser, go to Extension page, enable `Developer mode` option
3. Click `Load unpacked`, open the cloned repo or extracted folder (now a new icon will be added to extension bar, pin it as you like)

## Usage
1. Open a quiz set page
2. Click extension icon to open popup (fast or slow depends on the device you are using), customize your download file name, file format, parts order, image option
3. Click 'Scrape & Download' and wait
